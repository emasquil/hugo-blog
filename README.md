# hugo-blog
Code for my personal site using Hugo.

## How to use
* Spawn a dev server: `hugo serve -D`
* Add a new post: `hugo new content/posts/name-of-the-post.md`
* Build the site: `hugo` -> this builds the site in the directory `public`
